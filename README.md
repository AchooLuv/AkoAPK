# 憧门新番 - `APK`安装包仓库

自正式版`1.0.0`之后，`PATCH`版本通过应用内热更新，`MINOR`版本和`MAJOR`版本提供新的`APK`安装包，并且只提供`arm64`的安装包：

- [x] `arm64`
- [ ] `armeabi-v7a`
- [ ] `x86`
- [ ] `x86_64`
- [ ] `universal`

正式版`1.0.0`及之前的版本：

- [x] `arm64`
- [x] `armeabi-v7a`
- [x] `x86`
- [x] `x86_64`
- [x] `universal`
